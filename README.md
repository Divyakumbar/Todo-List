# 📝 To-Do List Web Application

A simple To-Do List web application built using Flask. This app allows users to register, login, and manage their daily tasks efficiently.

## 🚀 Features
- User Registration & Login
- Add Tasks
- View Tasks
- Mark Tasks as Completed

## 🛠️ Technologies Used
- Python (Flask)
- HTML
- CSS
- SQLite

## ▶️ How to Run
1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
